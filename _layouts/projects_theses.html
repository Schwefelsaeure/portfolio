Bachelor's thesis

During my Bachelor's thesis at the Infineon Technologies AG, I developed an "Evaluation Kit" to acquire pressure data from barometric air pressure sensors. The "Evaluation Kit" consisted of a microcontroller firmware in C and a GUI in C#/.NET.

See the presentation slides for a short overview or the Bachelor's thesis for more information.

Master's thesis

During my Master's thesis at the Fraunhofer Institute ESK, I developed a Linux kernel module to acquire GPS data with real-time requirements for Car2X scenarios. The Linux kernel module was implemented in C and gave me a wonderful insight in the well-thought-out and elaborated architecture of the Linux kernel.

See the presentation slides for a short overview or the Master's thesis for more information.

  • C
  • Linux Kernel
  • Scheduling

PhD

After years in the embedded software world, I got the chance to do a PhD in the field of crowd modeling and simulation. Thank you very, very much to Prof. Dr. Köster and Prof. Dr. Bungartz for this unbelievable opportunity to widen my horizon. I conducted an own crowd experiment and integrated social psychological aspects in our crowd simulator Vadere to make crowd gatherings safer.

See the PhD defense slides for a short overview or the PhD thesis for more information. The source code of our open source crowd simulator Vadere can be found on LRZ GitLab.

  • Java
  • Python
  • JUnit
  • pandas
  • matplotlib
  • GUI
